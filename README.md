# Nikkei 225 Topological Data Analysis (TDA)

This project visualizes the structural differences between the 1989 Asset Bubble and the 2026 Market using Persistent Homology.

## 📊 How to Run the Analysis
You can run the analysis using the provided Python script:
1. Open Google Colab or any Python environment.
2. Run the code from `analysis_script.py` to generate the persistence diagrams.

## 🧠 The Concept: Steel Framework vs. Bubble Ghost
- **2026 (Modern):** A rigid, high-persistence **"Steel Framework"** supporting the ¥53,123 reality.
- **1989 (Historical):** A decaying, low-persistence **"Bubble Ghost"** that lacked structural integrity.
